# SmartPay Manager - Project Blueprint

## Purpose
Build a web app for managing accounts payable and receivable with products and services.

## Target Users
Small business owners, accountants.

## Core Modules
1. Customers
2. Vendors
3. Products & Services
4. Invoices
5. Bills
6. Payments
7. Dashboard

## Workflow
Customer → Invoice → Payment → Ledger update
Vendor → Bill → Payment → Ledger update

## Database Tables (Planning)
- Customers
- Vendors
- Products & Services
- Invoices
- Invoice Items
- Bills
- Payments
